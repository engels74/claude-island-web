FAQ

Common questions

How do I install Claude Island?

Download the DMG file, open it, and drag Claude Island to your Applications folder. Since the app is not notarized, you'll need to right-click the app and select "Open" on first launch, then click "Open" in the security dialog.

Why does macOS show a security warning?

Claude Island is not signed with an Apple Developer certificate (which costs $99/year). This is common for open-source apps. Right-click → Open bypasses the Gatekeeper warning safely. You only need to do this once.

What macOS version do I need?

Claude Island requires macOS 15.6 (Sequoia) or later. The app uses modern SwiftUI features that are only available in the latest macOS versions.

Does Claude Island auto-update?

Yes! Claude Island includes Sparkle for automatic updates. You'll be notified when new versions are available, and can update with one click.

Is Claude Island open source?

Yes, Claude Island is fully open source under the MIT license. You can view the source code, contribute, or fork it on GitHub.
